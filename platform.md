* admin :

  - see his subjects

  - create lectures
  - add files & videos in lecture
  - create exam
  - receive exam solution and send grades
  - create questions bank 

* student can :

  - see his subjects
  - see every subject details 
  - access an exam and send solution
  - download lecture resources
